QtTextEditor

A lightweight, functional Notepad application built using C++ and the Qt Framework. This project demonstrates the implementation of file handling, clipboard operations, and a standard GUI application lifecycle using Qt Widgets.

Features
Based on the core logic, the application supports the following functionalities:

File Operations:
    * New: Clear the workspace for a new document.
    * Open: Browse and load text files from the local file system.
    * Save / Save As: Persist text data to local storage.
Editing Tools:
    * Cut, Copy, Paste:Full clipboard integration.
    * Undo / Redo: Text history management.
Interface:
    * Clean, resource-based icon toolbar.
    * Responsive text editing area.

Tech Stack

* Language:C++ (C++17 standard)
* GUI Framework: Qt 6 (Widgets Module)
* Build System: QMake
